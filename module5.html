What This Module Teaches

This module breaks Section 2 into two clear production lanes: CAD evidence (Part B) and spreadsheet material evidence (Part C).

  • Create CAD outputs that match drawing and standards requirements.
  • Create a computer-generated material list with cost totals.
  • Produce a pie or column chart showing total cost per item.
  • Submit screenshots/files that are easy for assessors to mark.
Assessment alignment: Section 2 Part B - Computer Aided Drawing (CAD), and Section 2 Part C - Material List.

Knowledge Check: Multiple Choice

1) Which output set is required for CAD evidence in Task 3?

One pictorial plus three orthographic views with dimensions.

2) In the material list, which formula is explicitly required?

Length x Quantity.

3) Which chart type is acceptable for the material-list cost output?

Pie or column chart.

4) Why is title block completion important in CAD submission?

It provides document identity and traceability.

Knowledge Check: Scenario Responses

Scenario 1: Your CAD model is complete but orthographic views are missing. Can this be submitted as complete evidence?

No. The required evidence includes the full view set; add orthographic views with dimensions before submission.

Scenario 2: The material list totals look high. What should you check first?

Verify units, quantity entries, and formula cell references before changing any source values.

Scenario 3: Why should CAD dimensions and material list values be cross-checked together?

It confirms consistency between design intent and procurement/output evidence, reducing assessor queries and rework.
