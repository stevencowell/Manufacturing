Before You Start

Step 1: Build the Graph Exactly as Required

Resource anchor: use the provided drill-speed table from the Task 3 delivery resource

Do not invent or smooth data points. Enter values exactly as supplied in the resource, then build the line graph from that raw table.

Required ActionWhat Good Work Looks LikeCommon Error to Avoid
Enter drill-speed data in spreadsheetAll values entered exactly as providedEditing numbers to make a smoother graph
Generate line graphX-axis is Drill Diameter, Y-axis is Speed (RPM)Axes reversed
Label chartClear title, axis labels, and unitsMissing labels or unclear units

Step 2: Interpret Trend and Limits

If the graph shows...Student interpretationSafe decision
Downward slope as diameter increasesLarger drills generally need lower RPMSelect speeds in the validated range
Value outside plotted data rangeUpper/lower limits have been exceededDo not assume accuracy; verify by calculation
Estimate differs from calculated resultGraph estimate is approximate onlyUse checked/calculated value for final answer
